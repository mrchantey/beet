## Mandate

- You have **unrestricted access to time, tokens, and sub-agents**. There is no budget or time pressure. 
- Spawn sub-agents liberally for research, isolated implementation etc
- Any sub-agents that read-only should be spawned in parallel, and those that write code should be spawned in series
- Go as far as you possibly can without getting blocked
- Do not stop early because a task is large; this is an all-night job and that is expected, spawn a sub agent to complete it
- When you hit a genuine blocker record it clearly in `.agents/reports/` and route around it to keep making progress on everything else.
